# 02_SPACING_JUSTIFICATION_RULES

## Global Spacing Scale
- Use `rem` for all values.
- `fluid-gap`: `clamp(1rem, 3vw, 2rem)`.

## Padding/Margin Normalization
- `container-px`: `px-4 sm:px-6 lg:px-8`.
- `component-my`: `my-4 sm:my-6`.

## Justification Standards
- **Headers**: `flex items-center justify-between`.
- **Mobile Buttons**: `w-full` if more than 2 buttons in a row.
- **Grids**: Use `grid-cols-1 sm:grid-cols-2 lg:grid-cols-3` pattern.
