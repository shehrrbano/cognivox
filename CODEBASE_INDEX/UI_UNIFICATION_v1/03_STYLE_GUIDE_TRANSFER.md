# Style Guide Transfer Rules

## 1. Typography
- **Font Family**: Inter, Roboto, or system sans-serif.
- **Section Headers**: Uppercase, tracking-widest, text-xs, font-bold, text-primary-500.
- **Body Text**: text-sm or text-base, text-gray-800, leading-relaxed.
- **Timestamps**: text-xs, text-gray-400.

## 2. Colors
- **Primary / Task**: Blue (`text-blue-500`, `bg-blue-50`, `border-blue-500`)
- **Risk / Warning**: Red (`text-red-500`, `bg-red-50`, `border-red-500`)
- **Decision / Success**: Green (`text-green-500`, `bg-green-50`, `border-green-500`)
- **Background**: White (`#FFFFFF`) to Off-White (`#F8FAFC`).
- **Graph Nodes**: Circular, 2px border matching semantic color, solid white background.

## 3. UI Components
- **Buttons**: Bright blue primary buttons with subtle hover states and crisp white icons.
- **Pills/Tags**: Padding `px-2 py-0.5`, rounded border or fully rounded `rounded-full`, uppercase text `text-[10px]` or `text-xs`.
- **Layout**: Dotted grid background for diagrammatic spaces, clean borders (1px solid `#E5E7EB`) dividing layout regions.

*(Will be implemented via Tailwind config and app.css by the Color/Spacing sub-agents).*
