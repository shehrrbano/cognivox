# 01 Internet Research - Cognitive Psychology

## Key Research Areas
### 1. Spatial Memory & the Method of Loci
- Users remember information better when it is linked to a persistent spatial location.
- **Action**: Nodes should maintain stable coordinates across sessions to support "spatial anchoring."

### 2. Cognitive Load Theory
- Excessive animation or "noisy" layouts increase extraneous cognitive load.
- **Action**: Use smooth damping in force simulations; group related nodes into clusters (chunking).

### 3. Color Psychology
- Functional color coding (e.g., Red for Risk) reduces search time.
- **Action**: Standardize node colors based on semantic type.
