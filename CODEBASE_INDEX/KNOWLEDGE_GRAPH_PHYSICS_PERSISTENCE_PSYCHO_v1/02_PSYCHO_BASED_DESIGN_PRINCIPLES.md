# 02 Psycho-Based Design Principles

1. **Persistent Landmarks**: The first node of a session should always center itself as a topological "North Star."
2. **Semantic Chunking**: Automatic clustering of TASKS vs DECISIONS vs RISKS.
3. **Minimal Distraction**: Animation should only occur during interaction or layout change.
4. **Stable Geometry**: Save and Restore every node's x/y coordinate for future visits.
