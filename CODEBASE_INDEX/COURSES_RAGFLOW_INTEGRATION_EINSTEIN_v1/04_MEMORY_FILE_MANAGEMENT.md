# 04. Memory and File Management

## File Lifecycle
1. **Upload**: Files are stored in the local resource folder and pushed to RAGFlow.
2. **Indexing**: Background processes track RAGFlow ingestion status.
3. **Deletion**: Two-stage deletion (Cognivox Store + RAGFlow Dataset).
4. **Expansion**: Seamlessly add new materials to an existing Course/Dataset.

## Management UI
- Per-course file browser.
- Metadata: Date added, Type, Size.
- Global Search integrates course contents.
